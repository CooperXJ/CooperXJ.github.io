安装dashboard

发表于 2022-01-07 更新于 2022-01-17 分类于 K8S Valine：
本文字数： 8.1k 阅读时长 ≈ 7 分钟

  1. 下载资源文件

    wget https://raw.githubusercontent.com/kubernetes/dashboard/v2.0.0/aio/deploy/recommended.yaml

  2. 创建资源

    kubectl get apply -f recommended.yaml

  3. 修改资源的svc为NodePort形式

  4. 因为一开始创建dashboard的时候默认创建的是一个最小权限的用户，因此我们需要自己去创建一个角色用户进行登录

    • 创建ServiceAccount资源

    • 创建角色绑定资源

      注意：cluster-admin是k8s自己创建的角色

    • 创建总的资源文件

    • 创建两种资源

      kubectl apply -f {file}

  5. 查看创建账户的token

    kubectl get secrets -n kubernetes-dashboard

    注意：需要将token进行base64进行解密
